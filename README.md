## BTC Converter with price chart

App developed with <b>Node</b> as BackEnd, <b>React</b> as FrontEnd and <b>Chart.js</b> library to represent the obtained data.<br>
Data obtained from https://www.coindesk.com/ API.<br>
<br>
<h3>Steps to run the App</h3>
<br>
1-Run `npm start` inside back folder to start the BackEnd.<br>
2-Run `npm start` inside front folder to start the React App.<br>
